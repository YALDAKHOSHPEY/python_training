YALDA NEON HEART
================

FASTEST WAY TO START
--------------------
1. Double-click START_YALDA_HEART.bat
2. The first launch may install Pygame automatically.
3. The heart window will open after installation finishes.

CONTROLS
--------
SPACE or click the heart : particle explosion
Click elsewhere           : sparkle burst
Move the mouse            : glowing particle trail
R                         : replay the opening
F                         : fullscreen
M                         : mute/unmute
ESC                       : close

CUSTOMIZE THE WORDS
-------------------
Right-click yalda_neon_heart.py and open it with Notepad.
Near the top, change:

NAME = "YALDA"

and edit the three lines inside MESSAGE_LINES.
Save the file, then run START_YALDA_HEART.bat again.

MANUAL COMMANDS
---------------
Install:
    py -m pip install pygame

Run:
    py yalda_neon_heart.py
