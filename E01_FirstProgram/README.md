# 🎬 Episode 01 —FirstProgram

---

## 🧠 Quick Review

- print() → show output on screen  
- input() → get input from the user  
- Variable → store data  
- type() → check data type  

---

## 📌 Important Note

Anything you get from input() is always stored as a string (text).
